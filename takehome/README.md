## Framing

Daymark-Anote is framed as an applied AI systems prototype for long-horizon research fellowship work. Instead of treating the fellowship calendar as a static schedule, it turns milestones, recurring rituals, and deliverables into a daily planning and review loop.

The central research question is:

> Can an AI-assisted planning system help research fellows translate long-horizon program goals into daily plans, reviewable progress traces, and actionable evaluation signals?
